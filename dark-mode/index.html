How Dark Mode Works

Class-Based Mode

Add the dark class to the html element to enable dark mode.

<html class="dark">

Media Query Mode

Alternatively, use media to follow system preferences.

darkMode: 'media' (default)

Configuration

This project uses class-based dark mode via darkMode: 'class' in tailwind.config.js

// tailwind.config.js
export default {'{'}'
darkMode: 'class',
// ... other config
{'}'}

This allows manual control via JavaScript and localStorage, with fallback to system preferences on first load.
