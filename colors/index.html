Color Palette Overview

Tailwind CSS comes with a comprehensive color palette designed for modern web design. Each color has 10 shades (50-900), from light to dark, enabling fine-grained color control.

Colors are available for backgrounds, text, borders, shadows, and more. You can also add custom colors to match your brand.

Color Usage Patterns

Text Colors

text-gray-900 - Primary text, headings

text-gray-600 - Secondary text

text-gray-500 - Tertiary text, hints

text-blue-600 - Links

Background Colors

bg-white - Main content
bg-gray-50 - Subtle backgrounds
bg-gray-100 - Hover states
bg-blue-50 - Alert backgrounds

Border Colors

border-gray-300 - Default borders
border-gray-200 - Subtle borders
border-blue-500 - Focus states

Customizing Colors

Extend Tailwind's color palette with your brand colors: 

// tailwind.config.js
export default {'{'}
  theme: {'{'}
    extend: {'{'}
      colors: {'{'}
        brand: {'{'}
          50: '#f0f9ff',
          500: '#0ea5e9',
          900: '#0c3d66',
        {'}'},
      {'}'},
    {'}'},
  {'}'},
{'}'};

Color Naming Conventions

Numeric Scale

50 (lightest) to 900 (darkest)

bg-blue-50 bg-blue-900

Primary Color

-500 is typically the main shade

bg-blue-500

Usage

Format: property-color-shade

text-blue-600
