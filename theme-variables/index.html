What is Theme Configuration?

Theme configuration allows you to customize Tailwind CSS to match your brand identity. Instead of using default utilities, you can extend the theme with custom colors, spacing, typography, and more.

The extend object in tailwind.config.js lets you add custom values while preserving Tailwind's defaults.

Configuration Structure

// tailwind.config.js
export default {'{'}
  theme: {'{'}
    extend: {'{'}
      // Add custom colors
      colors: {'{'}
        brand: {...},
        accent: {...},
      {'}'},
      
      // Extend spacing
      spacing: {'{'}
        '13': '3.25rem',
        '15': '3.75rem',
      {'}'},
      
      // Custom font sizes
      fontSize: {'{'}
        'xs': ['0.75rem', ...],
      {'}'},
      
      // Custom border radius
      borderRadius: {'{'}
        'xl': '1rem',
      {'}'},
      
      // Custom shadows
      boxShadow: {'{'}
        'premium': '0 25px 50px...',
      {'}'},
      
      // Custom animations
      animation: {'{'}
        'fade-in': 'fadeIn 0.5s...',
      {'}'},
      keyframes: {'{'}
        fadeIn: {'{'}
          '0%': {'{'}opacity: '0'{'}'},
          '100%': {'{'}opacity: '1'{'}'},
        {'}'},
      {'}'},
    {'}'},
  {'}'},
{'}'};
